# G29. User Manual

---

## 1. Panduan Umum Login

1. Buka browser dan masukkan URL sistem (misal: https://sis.smpit.sch.id).
2. Masukkan username dan password sesuai peran.
3. Klik tombol **Login**.
4. Jika lupa password, klik **Lupa Password** dan ikuti instruksi.

## 3. Tips & Troubleshooting

| Masalah | Solusi |
| --- | --- |
| Tidak bisa login | Periksa username, caps lock, hubungi admin jika lupa password |
| Halaman lambat | Refresh, clear cache, atau coba di jam tidak sibuk |
| Data tidak tampil | Pastikan tahun ajaran aktif sudah dipilih |
| Cetak PDF error | Periksa popup blocker, gunakan browser Chrome/Edge |
| Akses ditolak | Pastikan Anda memiliki hak akses ke menu tersebut |
