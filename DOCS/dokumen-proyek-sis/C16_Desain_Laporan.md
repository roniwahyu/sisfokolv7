# C16. Desain Laporan

---

Berikut contoh layout dan deskripsi 6 jenis laporan utama.

## 1. Laporan Rapor Semester

| Bagian | Keterangan |
| --- | --- |
| Header | Logo sekolah, nama sekolah, alamat, tahun ajaran, semester |
| Identitas | NIS, nama siswa, kelas, wali kelas |
| Tabel Nilai | Mapel | UH | PTS | PAS | Akhir | Predikat |
| Absensi | Sakit, Izin, Alpha, Hadir |
| Deskripsi | Deskripsi sikap/akhlak dan capaian kompetensi |
| Footer | Tanda tangan wali kelas & kepala sekolah |

## 2. Rekap Nilai per Kelas

| Bagian | Keterangan |
| --- | --- |
| Filter | Kelas, semester, tahun ajaran |
| Kolom | NIS, Nama, Mapel, UH, PTS, PAS, Akhir, Predikat |
| Footer | Rata-rata kelas, jumlah siswa |

## 3. Rekap Absensi Bulanan

| Bagian | Keterangan |
| --- | --- |
| Filter | Kelas, bulan, tahun ajaran |
| Kolom | NIS, Nama, H, S, I, A, % Kehadiran |
| Footer | Total per status kehadiran |

## 4. Laporan Pembayaran SPP

| Bagian | Keterangan |
| --- | --- |
| Filter | Bulan, kelas, status (lunas/belum) |
| Kolom | NIS, Nama, Kelas, Tagihan, Dibayar, Status, Tanggal Bayar |
| Footer | Total pembayaran, total tunggakan |

## 5. Laporan Keuangan Harian/Bulanan

| Bagian | Keterangan |
| --- | --- |
| Filter | Tanggal / bulan |
| Kolom | Tanggal, Jenis, Jumlah, Metode, Keterangan |
| Footer | Total pemasukan, total per jenis pembayaran |

## 6. Dashboard Executive untuk Kepala Sekolah

| Bagian | Keterangan |
| --- | --- |
| Kartu Ringkasan | Jumlah siswa, guru, kelas, total pembayaran bulan ini |
| Grafik | Kehadiran bulanan, perbandingan pembayaran SPP |
| Tabel | 5 siswa dengan absensi terbanyak, 5 tagihan terbesar |
| Tren | Grafik perbandingan nilai rata-rata per semester |
