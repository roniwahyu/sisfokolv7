## Cara Membaca

- Buka `index.md` untuk melihat daftar dokumen dan tautan.
- File `.md` dapat dibaca langsung atau di-convert ke PDF/Word dengan pandoc.
- File `C15_Desain_Antarmuka.html` dapat dibuka di browser untuk melihat wireframe.

## Konteks

- Jenis sekolah: SMP
- Stakeholder: Kepala Sekolah, Wakil Kepala Sekolah, Tata Usaha, Guru, Wali Kelas, Siswa, Orang Tua, Bendahara.
- Metodologi: Dokumen mengikuti standar rekayasa perangkat lunak dan best practice ITIL/ISO untuk manajemen sistem informasi.
