### Prinsip (ADR-007)
- InnoDB, utf8mb4_unicode_ci
- PK BIGINT UNSIGNED AUTO_INCREMENT
- FK + ON DELETE/UPDATE
- Standar: timestamps + soft_deletes + created_by/updated_by
- Tipe sesuai domain (decimal uang, tinyint nilai)
- tenant_id + composite index semua tabel domain
- Helper migration macro: `$table->tenantAndAuditColumns()`

### Normalisasi utama vs SISFOKOL
- PK: varchar(50) MD5 ‚Üí BIGINT
- Uang: varchar(15) ‚Üí decimal(15,2)
- Snapshot siswa/item di transaksi ‚Üí FK ke master
- Orang tua: passwordx_ortu di m_siswa ‚Üí tabel orang_tua + pivot
- Tahun ajaran aktif: hardcoded session ‚Üí tenant_settings key-value
